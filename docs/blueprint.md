# **App Name**: Doorstep Delights

## Core Features:

- Dessert Catalog: Browse a visually appealing catalog of desserts with descriptions, images, and pricing.
- Shopping Cart: Add and manage items in a shopping cart.
- Order Customization Tool: AI tool suggesting the best cake options with decorations or flavor for events depending on information provided such as 'gender of the celebrant, favorite hobbies'
- Checkout Process: Secure checkout with credit card payment via Stripe integration.
- Delivery/Pickup Options: Choose between doorstep delivery or in-store pickup at checkout.
- Order Tracking: Track the status of your order.
- Admin Panel: Manage products, orders, and customer information via an admin panel.

## Style Guidelines:

- Primary color: Warm, inviting rose-pink (#F472B6), evoking sweetness and comfort.
- Background color: Very light, desaturated rose-pink (#FDE8F3) for a soft, pleasant backdrop.
- Accent color: Lilac (#A855F7), providing contrast and a modern feel to buttons and highlights.
- Body and headline font: 'Alegreya', a serif with an elegant, intellectual feel, is a good choice for headlines or body text.
- Use clear and recognizable icons related to desserts, shopping cart, delivery, and pickup.
- Clean and spacious layout with high-quality dessert images to entice customers.
- Subtle animations on product hover and cart updates to enhance user engagement.
